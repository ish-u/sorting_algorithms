# Sorting Algorithms

### GUI App made using Tkinter which displays the Graph of Comparisons v/s Size for 100 Test Cases for Different Sorting Algortihms

To Run the App :
```
python main.py
```

#### Sorting Algorithm
    1. Selection Sort
    2. Bubble Sort
    3. Instertion Sort
    4. Merge Sort
    5. Quick Sort
    6. Heap Sort
